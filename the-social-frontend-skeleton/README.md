# The SOCIAL — Frontend Skeleton

This repo is the frontend scaffold for the SOCIAL network project.
It follows the documented flow:
- React Hook Form for client validation
- TanStack Query for REST data/caching
- Axios with JWT for standard HTTP requests
- SignalR client for realtime chat/presence
- Tailwind CSS for UI styling

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```
